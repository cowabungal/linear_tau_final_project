## Запуск

```
- открыть init_constants.m. Нажать Run.
- открыть main.slx. Нажать Run.
- открыть DataInspector. Выставить интересуемые сигналы.
- Поверить, что мы старались
- Убедиться в идеальности системы (необязательно)
```
